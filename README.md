# Defence Sector Stock Market Analysis

A data analysis project that examines historical stock market data of Indian defence companies using Python and R.

## Features

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Central Tendency Analysis
* Dispersion Analysis
* Line Chart Visualization
* Correlation Heatmap

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* R
* dplyr

## Output Files

* `combined_defence_data.csv`
* `statistics_results.csv`
* `line_chart.png`
* `heatmap.png`
